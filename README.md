# Markdown Preview

A markdown editor with live preview.

---

## Features

- Split, editor-only, and preview-only views
- Scroll sync between panels
- Dark and light themes
- Word count, character count, reading time
- Copy as Markdown, HTML, or plain text
- Resizable panels
- Responsive layout

---

## Setup

```bash
pnpm install
pnpm run dev
```

---

## Shortcuts

| Action | Shortcut |
|--------|----------|
| Split view | `Ctrl + \` |
| Editor only | `Ctrl + Shift + E` |
| Preview only | `Ctrl + Shift + P` |

---

## Stack

React, TypeScript, Vite, react-markdown, react-split.
